# Contributing

This repository is public for reference and forking. Direct write access is not open.

If you have a suggested improvement, open a Pull Request and I'll review it.

## Guidelines

- Keep `create-skill.md` focused on distilling sessions into skills. No scope creep.
- Improvements to how the workflow is extracted from a conversation, how the SKILL.md is structured, or how edge cases (thin sessions, conflicting approaches) are handled are welcome.
- The draft-and-confirm step is intentional — don't remove it.
